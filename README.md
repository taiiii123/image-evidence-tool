# エビデンス作成ツール

画像を整理してExcelでエビデンス資料を作成するためのVue 3アプリケーションです。

## 特徴

- 📁 複数画像の一括アップロード
- 🎨 ダークモード対応
- 🔄 ドラッグ&ドロップで並び替え
- 💬 各画像にコメント追加
- 📊 Excel形式でエビデンス出力
- 🖼️ 画像のライトボックス表示
- ⚙️ カスタマイズ可能な画像サイズと配置

## セットアップ

### 前提条件

- Node.js 16以上

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# プロダクションビルド
npm run build

```

## プロジェクト構造

```
src/
├── components/           # Vueコンポーネント
│   ├── TheHeader.vue    # ヘッダーとコントロール
│   ├── ImageGrid.vue    # 画像グリッド
│   ├── ImageItem.vue    # 個別画像アイテム
│   ├── Toast.vue        # トースト通知
│   ├── Modal.vue        # モーダルダイアログ
│   └── ThemeToggle.vue  # テーマ切り替えボタン
├── composables/         # 再利用可能なロジック
│   ├── useToast.js      # トースト通知機能
│   ├── useTheme.js      # テーマ管理
│   └── useExcelExport.js # Excel出力機能
├── assets/
│   └── styles/
│       └── main.scss    # メインスタイル
├── App.vue              # ルートコンポーネント
└── main.js              # エントリーポイント
```

## 使い方

1. **画像を選択**: 「📁 画像を選択」ボタンから画像ファイルを選択
2. **並び替え**: ドラッグ&ドロップで画像を並び替え
3. **コメント追加**: 各画像にコメント(説明・備考)を記入
4. **Excel出力**: 「📥 Excelエビデンス出力」でダウンロード

## 技術スタック

- **Vue 3**: Composition API
- **Vite**: ビルドツール
- **SCSS**: スタイリング
- **ExcelJS**: Excel生成
- **SortableJS**: ドラッグ&ドロップ
- **PhotoSwipe**: 画像ライトボックス

## カスタマイズ

### 画像設定
- 画像幅・高さ: Excel出力時の画像サイズ
- 行間: 画像間のスペース
- 左余白・上余白: Excelシートの余白
- 連番: 画像に番号を付ける

### 並び順オプション
- ファイル名順(昇順/降順)
- 更新日時順(新→古/古→新)
- ファイルサイズ順(大→小/小→大)
